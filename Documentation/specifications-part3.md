# ID Card
## Electronic format
- is in the format `AA12345BB` with:
    - **2** letters
    - **5** digits
    - **2** letters

## Paper format
- is in the format `AA1234567` with:
    - **2** letters
    - **7** digits

# Passport
- is in the format `AA123456789` with:
    - **2** letters
    - **9** digits
